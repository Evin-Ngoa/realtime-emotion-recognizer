# Realtime Emotion Recognition Application with Pusher + Tensorflow.js + Vue

![demo](./demo.gif)

## Project setup

```
yarn add
```

### Run frontend application

```
yarn serve
```

## Run the backend server

```
cd server
yarn add
yarn start
```

### Compiles and minifies frontend for production

```
yarn build
```

### Lints and fixes files

```
yarn lint
```
